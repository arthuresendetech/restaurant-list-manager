# Leitor e Manipulador de Lista de Restaurantes
[🇺🇸 English](README_EN.md)

Esse projeto foi criado por [@AugustoFrancoDev](https://github.com/AugustoFrancoDev) e [@arthuresendetech](https://github.com/arthuresendetech).  
O programa foi desenvolvido em **C++** e consiste em ler os dados do arquivo **`restaurantes.csv`** sobre restaurantes com estrela Michelin e permitir manipular esses dados diretamente pelo programa.

## Funcionalidades
  1 - A lista pode ser ordenada através de funções quicksort.  
  2 - Você pode buscar restaurantes através de busca binária.  
  3 - É possível criar novos restaurantes na lista.  
  4 - Também é possível remover restaurantes.  
  5 - Todas as mudanças podem ser salvas no arquivo **`restaurantes.csv`**.

Basta compilar o arquivo **`projeto.cpp`** e executá-lo na mesma pasta que o arquivo **`restaurantes.csv`**.
