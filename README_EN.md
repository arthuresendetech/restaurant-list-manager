# Reader and Manager of Restaurant List
[🇧🇷 Português](README.md)

This project was developed by [@AugustoFrancoDev](https://github.com/AugustoFrancoDev) and [@arthuresendetech](https://github.com/arthuresendetech).  
The program was developed using the **C++** language and consists of reading data from the **`restaurantes.csv`** file about Michelin-starred restaurants and managing this data through the program.

## Features
1 - The list can be reordered using a quicksort function.  
2 - You can search restaurants using binary search.  
3 - It's possible to create and insert new restaurants into the list.  
4 - You can remove restaurants.  
5 - All changes can be saved to the **`restaurantes.csv`** file.

Simply compile the **`projeto.cpp`** file and execute it in the same directory as **`restaurantes.csv`**.
